# 国赛常见题型总览

## 常见题目特征
这类问题常包含评价、预测、优化、分类、仿真、网络和混合建模等信号。题面可能给出多张表、多个对象或连续多问，需要先判断每问的目标。

## 常见小问形式
第一问多为数据处理或规律描述；第二问通常建立核心模型；第三问进行预测、评价或优化；最后一问常要求改进、推广或建议。

## 适合模型类别
先识别题型，再为每个小问选择方法类别。模型选择应优先满足数据可用、结果可解释和论文可表达。

## 数据需求
需要题面文本、字段说明、样本对象、时间或空间字段、评价指标、约束条件或网络边表。缺少数据时应记录假设并给出降级方案。

## 输出要求
输出包括中间表、模型参数、排名、预测值、最优方案、图表和对每个小问的明确回答。

## 容易出错的地方
题型判断过早、多个小问共用不一致指标、只写算法不回答问题、没有说明数据预处理和稳定性。

## 推荐解题流程
1. 提取背景、数据和小问。
2. 判断每问题型和输出。
3. 生成候选模型并打分。
4. 执行可复现代码。
5. 用图表解释结果。
6. 做误差、灵敏度或鲁棒性分析。
7. 回到题意组织论文。

## 与论文结构的关系
题型识别结果决定问题分析、模型建立、求解和结果分析的章节组织。每个小问应有独立模型说明和结论。
