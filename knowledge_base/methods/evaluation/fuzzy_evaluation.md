# 模糊综合评价

## 方法类别
评价类模型，适用于多指标评价、综合排序、权重计算、方案比较、风险评价。

## 关键词
模糊评价、隶属度、评价等级、模糊矩阵、综合评价

## 适用问题
评价标准存在模糊边界，如风险等级、满意度、质量等级。题面通常包含多个对象、变量或约束，需要把数据处理、模型计算和结果解释连接起来。

## 不适用问题
等级划分缺少依据或隶属函数随意设定时不宜使用。若数据质量不足，应先做缺失值、异常值和口径一致性检查，再决定是否使用该方法。

## 输入数据要求
- 需要明确样本、对象或节点的定义。
- 需要字段含义、量纲和方向说明。
- 需要区分数值型、类别型、时间型或网络型数据。
- 若涉及监督学习，需要目标变量或标签。
- 若涉及优化，需要目标函数、决策变量和约束条件。

## 输出结果
- 模型参数、得分、类别、路径、预测值或最优方案。
- 可形成结果表、排序表、误差表或方案对比表。
- 可生成隶属度热力图、等级分布图、综合评价表。
- 论文中应解释数值含义、适用范围和可能偏差。

## 基本步骤
1. 明确问题目标和变量定义。
2. 清洗数据，统一量纲、方向和缺失值处理规则。
3. 构造模型输入矩阵或网络结构。
4. 按方法公式计算核心指标或求解目标。
5. 输出结果表和关键图表。
6. 做误差、灵敏度或鲁棒性检查。
7. 将结论转化为面向题目的解释。
